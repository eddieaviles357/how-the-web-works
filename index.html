How the Web Works

  • What is HTTP? HTTP stands for Hyper Text Transfer Protocol. I'ts a protocol used by web browsers to send information.
  • What is a URL? URL stands for Uniform Resource Loacator. It is a Srting that has the protocol, Hostname, port, resource, queries.
  • What is DNS? DNS stands for Domain Name System. It is a system to associate human readable names( URL's ) to IP addresses.
  • What is a query string? A query string are additional arguments that can be used by a web server for arguments. It starts with ? then key=value and the & is used to chaing additional queries
  • What are two HTTP verbs and how are they different? 2 types of HTTP verbs are GET and PUT. GET verb is used to retrieve read only files and PUT verb is used to modify something.
  • What is an HTTP request? An HTTP request is usually done by a client ( Browser ) and it's used to request files. A request has headers like Accept.
  • What is an HTTP response? An HTTP response is usually sent by a web server to the client an example is JSON data. A response has headers like Status.
  • What is an HTTP header? Give a couple examples of request and response headers you have seen. An HTTP headers are information sent to the client or server about the packet. Some examples are Accept, User-Agent, Content-type.
  • What are the processes that happen when you type “http://somesite.com/some/page.html” into a browser? When a URL is entered in the browser it gets parsed by the browser and figures out what protocol to use, what hostname address to use, what port to use, what resource to get, what verb to use. It gets sent in a packet with your IP address and destination IP address to the internet, which are a bunch of computers, servers.
